# Log-File-Reader
This Python script reads a log file and counts the number of INFO, ERROR, and WARNING messages. It displays the results in a formatted summary table.

## Features
- Reads log files
- Counts INFO logs
- Counts ERROR logs
- Counts WARNING logs
- Displays summary table

## Requirements
- Python 3.x

## Installation

```bash
git clone https://github.com/vamsi-2005-cs/Log-File-Reader.git

cd Log-File-Reader
```
## Usage
1. Place `app.log` in the project folder.
2. Run the script: python log_analyzer.py

## Code Explanation
- `Counter()` stores counts of log levels.
- The script reads the log file line by line.
- It checks for INFO, ERROR, and WARNING messages.
